    第4期 无忧时间

    April 5, 2024
    本文总计 0.02k 字, 阅读约需要 0.23 分钟
    2024
    周刊, 2024

    清明放假，第二天了才想起来上次写周刊还是上月初，实在是不忍心在拖下去了来补充一下。 三月还是一样待在家里，没怎么出门，周刊也没有写= - =！实际上也不知道干嘛了。 吃 # 前几天必胜客有自助餐，等我打开app看，盐城地区已经没有票了，后来打电话要到一个到店支付名额遂去。真的吃爽了（可惜那天路考科目挂了。。。。第一次死在了起步熄火了扣10分坡道起步扣10分，第二次走到绕桩被人工判定负，说是起步没按喇叭导致流程还没开始又就这样两条命都没了，十天后才能继续考，今天看了一下应该清明结束就可以去重考了。。。。 清明祭祖 # 今年祭祖任务比较重，因为需要首先要去把祖先牌接到家里，明年祭祖就轮到我家了。另外因为队里要拆迁了，家族里人今年都赶回来烧纸，到公墓里人比往年要多。 往年都是爸爸和我两个人，最多再加上妈妈，今年又去接上了大爷爷、大奶奶，二老下面没有宝宝了，本来有个坤子大爷，年前的时候杀了人被抓起来，枪毙了。之后每年回去要先去接上他们二老一块回去了。 银元被偷 # 老房子梁上一直有用红绿布包着的两块银元，今年回去发现被偷了 奶奶走后，老房子就一直没有住人，村里xun大爷爷（一直也不知道是哪个xun）说家里想借客厅放东西索性就把家门钥匙放在大爷爷那里。清明回去发现，之前一直挂在梁上用红绿布包着的银元没了。诶，唯一的钥匙在大爷爷那里，妈妈那里甚至都没有钥匙，再说平常溜门撬锁的人怎么注意到哪个梁上呢？ 大爷爷也没说什么，就说了我总不能拿吧。。。诶，不到要拆房子时候，平常没人注意到放在梁上用来镇宅的银元。 家里还陆续丢了铜炉，小时候冬天奶奶都用这个铜炉来装炉灰取暖，爸爸说可能还有什么也没注意到。 电影 # 平常无事可看。蛮牛逼，最后教堂那段屠杀戏看得真爽，阮经天的演技撑起了这部戏。 这部还是公映的第一天去看，看了豆瓣上评价却是拉了，人物没立住。倒是看电影的时候隔壁座看到皇帝跪倒在保罗前面时候嘀咕了句：“升官发财死老婆”真的逗乐了。 其实还看了、一年又一年、黑冰、羊城暗哨、沙家浜都蛮一般。我可能是先看了《过把瘾》里的王志文，再看黑冰就真的看不下去了，一年又一年就完全是看作女和直男斗法。 另外可以说的是杨德昌导演的麻将，可惜看到的是巨糊的片源，最后感觉，骗人的人骗来骗去最后还是要骗到自己头上。

    第3期 回村晒稻

    March 11, 2024
    本文总计 0.01k 字, 阅读约需要 0.14 分钟
    2024
    周刊, 2024

    拆迁 # 今年刚开始，队上的会计就挨家来问，村里要拆迁了，家里是否需要宅基地以及南边的坟是否要移。村里的干部希望我们能配合拆迁，当然我也很高兴，家里厨房上的钢筋已经裸露出来了，按照二姨夫的话，得亏现在政府来帮你拆，不然再过几年谁知道房子倒不倒。至于爷爷的坟队里意思如果家里人实在不行离开村里，就在埋在田里，不用在跑到很远的陵园里，那里的一块地方说是过了八年之后还需要缴钱才能继续呆。 老房子 # 老房子正门 87年建的房，我问我爸那时候建房子花了多少钱，他笑着说当时砖头全是自己烧的没花多少钱就建起来，到现在城里动一动就是上百万。。。唯一花了钱的是门口的瓷砖，要买齐对应的图案在那个时候很难，特地跑到水浒大桥去买。门口的龙还有这次回去有拍了张照片，家里的忘记拍了，下次需要补上！现在看，我爸年轻时候意气风发的时候建的这个房在队里还算蛮好，三间瓦房变成了水泥房。 电影 # 妇女节那天，正好是星期五没事，正好去看了《沙丘2》。诶，看完后悔了38块钱的票钱了。。。。。之前看上部有很多宏大的叙事刚开始展开，以为这部会很好看，结果就多了点大场面的CG，人物性格完全没立住甚至不知道为什么保罗就会和契妮分开，中间的权谋戏份完全没有，决斗戏也好敷衍。

    看剧名本来兴致冲冲的点进去了，看了两集遂弃，期待是像电影版那样紧张刺激中带点小幽默结果给我整了个文艺范十足的谍战。。。遂弃 被封 # 闲鱼账号被封了，说是因为我发布领养狗那条信息的配图包含色情。。。就很难理解，文本信息是盐城当地是否有没有时间照顾的小狗我想领养一只。不知道是否被理解成了肾虚？？或者是某些LGBT元素？就被永封了。。。 申诉 # 闲鱼的申诉机制类似于，系统管理员因为你这个帖子有某些敏感词锁着，他来判定是否需要处罚以及需要怎样的处罚，这个过程中你啥都不能做唯一可以操作的是可以删除自己的帖子 看见自己帖子进了小黑屋，赶紧删除 被锁后，有一次的申诉机会，认真回应（我就随便截了张最近聊天的图上传，就被ban了 最后大招是联系闲鱼客服：9510222，找到人工态度强硬申诉（我态度强硬，说是联系专员了 最后还是锁住了，那就只能重新换号了，本来想着去互联网法院起诉但是想着我的配图可能有点问题？不知道肾虚算不算色情？遂弃 理财 # 2024年少碰大a，过年那几天看100日元对人民币汇率跌倒了4.9–4.8这个区间，比之前要少不少，遂兑换了五万人民币想着等日元升值再换回来。 驾照 # 去报名参加了摩托车的考试，这个我没办法加速，要么可以选择去驾校报名800元 or 自费 135元走盐都车管所报名即可，窗口辅警建议学习一下挂档+坡道停车即可。

    第1期 过年记忆

    February 14, 2024
    本文总计 0.03k 字, 阅读约需要 0.27 分钟
    2024
    周刊, 过年

    这张照片是今年为数不多下雪那天到窗口拍下。 现在是下午七点，刚吃过饭坐在电脑前面想着来记录一下24年的春节，怕春节的记忆就这样滑走。初五的日子里外面都是鞭炮声我啥都想不起来，过年这段时间的记忆就真的快要没了。 年前准备 # 剪头 # 在镇子上理发店里排队等着理发 腊月二十八就停下来了，过年的习惯就是去剪个头、洗个澡、买新衣服、给老人送压岁钱。搬新家之后还没在这附近找到一家合适的理发店，想着等爹妈之后一起去张庄那里剪头顺便去把那里楼下车库给收拾一下，去楼道把车库去年小半年的用水用电来统计出来，把钱和楼上租客结清还有隔壁领居借钱后还有2k说是等过去给我现金，后面一直等到下雪那天还去把钱拿回来。就在镇子上找了家理发店和我爸一起去剪个头。 大姑去世三年 # 大姑已经走了三年了，留代哥喊到了年前把事情做了喊了和尚来放焰口。 好像对大姑的记忆真的很少，她比我爸大八岁这样就很早嫁出去离开了村子，嫁到了隔壁村子后大姑伯（方言：大古柏）很早就走了大姑又改嫁到县里，离我家很远平常也没怎么来看过年见面机会也不多，奶奶在的时候还可以约着一起去给奶奶送节礼。 大姑走的时候也很老了，年纪才五十几岁但是整个头发都白了，听我爸讲解刨的时候发现大姑的肝脏也已经硬化了说是就算没被车撞到，回到家里也就只有一两年的命了。 留代哥哥把大姑的死了的赔偿金立刻搬出来去城西南买了套房这样就可以给小侄女上学。 暖冬 # 家里的茶花十月搬来的时候种在阳台，前几天看要开了 这个冬天有一波强降雪，外婆拿塑料薄膜来罩着青菜，奶奶种田的手艺真好 体重 # 拍摄于 2024.2.14 体重巅峰，过年一直在吃同时因为游泳馆闭关一直没有训练 过年结束了，需要尽快把体重给控制住，到了夏天不然又得去买衣服近几年都是一年换一次衣服，上年的衣服拿出来还是新的，不过已经穿不上了 = - =！
