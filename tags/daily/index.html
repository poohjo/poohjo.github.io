    第6期 小猫进家

    April 16, 2024
    本文总计 0.03k 字, 阅读约需要 0.3 分钟
    2024
    daily

    诶，一个人在家太无聊了，经常一坐电脑前半天，想着有时间为啥不养只猫在家呢？？ 打开闲鱼搜，看见一个只3.25刚出生的，而且还是在老家镇上，之前一只琢磨有时间回去打印下户口本，这玩意现在还得去你户口所在地派出所。。。= =！ 礼拜一联系的，约了周六去带回来。3.25-4.20，出生25天。 诶，爸就很担心它拉屎问题，比如万一学不会猫砂，拉在了沙发中间？？？拉完了怎么擦屁股粘的那点呢？妈那头就简单的害怕，说年轻的时候迫不得已养了一只来抓老鼠，之后没老鼠了猫就送人了，我估计只要威胁她收拾家里加要她扔了她房间里那些乱七八糟东西（她那房间诶也不能称房间了，和垃圾堆比就多了张床，老妈都不去睡了，到我这就去老爹房间睡）能接受，后面就看小猫的了。 而且有个坏消息，爸那边打过电话了，说五一回来让我带他出去走走？？？他天天在外面跑还缺我带他嘛，再说五一节出去玩那不是看人海，弃！ 到时候爸妈回来，待在家里，一猫应该也才到家七八天，希望小猫能快速抓住爸妈心，至少别乱拉屎被逮到。。。 上次调侃我爸，家里真董事长走丢了，他一点也不心疼。这也没过多久，就来新董事长了而且还是要带进家，之前那位都在放车库外，到点回家吃饭就行。 那两位就当没看见，你喂你的我看不见就行。 准备 # 1、笼子。本地闲鱼联系了一个15自提，有点掉漆，回来擦了下完美！ 2、航空箱。 这玩意估计只有接小猫时候能用到，本来想找个纸箱得了，无奈怕乱动，又联系了本地自提25元人民币完美！ 3、猫粮。搜了会，好复杂，从全价到三拼，从猫吃什么到猫讨厌什么，选择半天，先买点网易的粮回来（240，4袋）好贵，这个要是让老爹老妈知道得心疼呀，这次死也不说价格，不然估计猫就只能喝粥了。后面考虑是不是能自制。毕竟超市鸡胸肉才6块，再拼点冻巴沙鱼，弄成小块应该也是美滋滋！ 4、猫砂，这个完全不懂，先搞点100斤回来，后面和送猫那家联系了他们家也没搞，就是让猫固定到家旁边排水道里去啦，自己也打算看小猫是否能学会用马桶。 5、玩具，刚出生应该没那么活泼吧，需要再买。 到家 # 左边：小橘 右边：傻豆 小橘：特别活波，乱动，走到哪都叫。唯一安静的时间就是它和傻豆一起睡觉。 傻豆：完全不怎么动，比小橘晚五天出生，到家也没叫，也不爱喝奶。。。 昨天去接猫，外面天气太差了路上刮着小旋风，天也是灰蒙蒙的。那家人就在秦南镇子里，一个小桥下去，约好了从桥头下去家里人来带我，七绕八绕就拐到了那家。 10只小猫围着躺在地上的小母猫，傻豆就很僵硬的看着其他猫喝奶，小橘就在那里打架，本来打算就带傻豆回来，那家女主说：“一只回去不得闹死啊，两只回去还做个伴” 回来一天了，还不知道傻豆是男是女？？它睡觉就缩着屁股、走路就压着尾巴（完全不像小橘，竖着尾巴，找机会要偷拍一下。。。 猫奴 # 小橘喝奶真的快，抱着针筒就喝。小橘今天下午突然在笼子外排尿了，好事！ 傻豆塞嘴里，完全不想喝？真怕它饿到。要不是都满一个月了，我都打算去买奶嘴了。唯一庆幸是这小姑娘（看菊花，暂定母的）排尿很正常，用湿纸巾碰了碰，就有小液滴流出。

    为什么我要写博客呢？ # 2023年在家躺了一年，感觉时间过得太快且没有任何事情被记录下来，从去年开始自己记忆力下降很快，常常下午就忘记了上午做了什么。脑袋经常性发木，持续时间还久。 24年就开始想着写日记，无奈写在纸上本就难检索，外加长时间没有写过字，写出的字太过鬼画符了，遂想着上网找地方写日记。试过在Word上，如果不新建一个doc，单文件过大且自己需要个类似于Tyora类似的目录结构，遂放弃。 在网上找了许久，找到个这位 「椒盐豆豉」的博客，尤其是这样的book结构太方便我来查看,但是对于前端代码真的一点不懂不知道怎么改，好在自己高中时有捣鼓过点Chrome浏览器,了解点网页是从html渲染来的，但是也就知道这个。后面遂通过F12键显示源文件，加ChatGPT帮助下，扒下来大概结构。 Hugo 安装 # 大概结构是有了，无奈只能本地访问html文件且格式还无法显示，乱七八糟的，可能是扒站方式不对。倒是这个时候，利用ChatGPT得知了Hugo的大概建站过程，同时需要感谢 「椒盐豆豉又双叒叕一篇小白个人博客选什么工具或平台」中有 Hugo建站过程,很重要的是留下了为什么选择Hugo的原因，直接避免了我选择Wordpress。 @ 椒盐豆豉 ：建站不可能三角.jpg 安装基本参考了 「小绵尾巴」中的建站过程，当时遇见蛮多问题，后续通过面向ChatGPT，总算是能有html大概结构出来，但是一直出现一个问题本地打开 html，出现乱码，打开console中一堆爆红，本来都打算放弃了真的因为不知道去哪搞，仔细看看了也很绝望大概就是file:///xxxx/xxxxx.svfg类似这样的爆红，这个时候就要吐槽ChatGPT 3.5 了，就只能提示什么file:///这里应该为file://这样的错误，当时就怀疑是不是哪里扒站的时候多了个斜杠，用Vscod全局搜索了下，没找到三个斜杠地方。。。 都开始试着用Notion来写日记了，无奈那个响应速度确实慢（可能是我主机不行，或者网络不太好。。 后面试着先搞GitHub Pages,文件上传后居然正常显示了，查看了下本来错误的地方现在变成https://就这样变成两个杠，解决。 工具有了，开始记录 # 这个时候看见了 「潮流周刊」这种样式，遂开始装修Hugo。诶，这个时候其实还没开始用Hugo记录几篇，markdown文本格式才刚上手写。但是无奈我引用的图片格式实在太丑，开始了解shortcode 参考的教程 # Hugo | 在 Stack 主题上可行的短代码们 Hugo Shortcodes 示例 博客重启 2.0 尚缺功能 # 能督促自己有记录习惯，暂时还没这个功能，也不知道怎么弄。 图片显示，写了三篇，图片显示全靠现查找 shortcode，大概样式需要重建一下，这个其实我只需要pc端打开不要丑到我真的受不了就行。 评论，这个功能后续再看吧。
