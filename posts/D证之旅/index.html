    D证考试（2024江苏盐城地区 #

    科目一，四。考试选取50道题目，最多错五题

    科目二，三 #

    点火（上次就死这里扣了10分 #

    • 戴头盔，戴好带证即可。
    • 上车同时先放手刹，转钥匙，点火。之后按完喇叭，开始人脸比对（进入考试环节
    • 捏离合，脚挂一档，慢慢松开离合（我就是死在这里，左手松开离合，右手没有转，熄火！

    单边桥 #

    • 这关我也不知道怎么过 但是考试也没掉下去，可能是桥比较宽，怎么走都走上去。。。
    • 第一块上窄桥前，目测前轮距离宅桥10cm？？？大概就是一只脚的距离
    • 下第一块，就赶紧转正车身，轮胎还是距离右边窄桥10cm

    过人型横道 #

    • 刹车，左右摇头

    绕桩 #

    • 进去前还是一样车身摆正，刚过第一个桩，大概10cm，向左打死。
    • 前轮距离边线15cm，前轮平行于边线走，然后再到右边刚过桩，再向左打死
    • 大概瞄点就这样

    坡道起步（扣10分 #

    这个倒是真的需要考。想想真在路上坡道，遇到前面一个溜车真的吓人👻

    这关也不知道怎么考，我上次考这关，也是莫名其妙扣十分。主要是这个考试 无论是考生，还是车管所都蛮随意，至少今年我们这边是这样的。。。上次去考试，我前面还有两个人，一人在考试一人在考试区域待考，辅警从办公室出来，把待考女生踢出了这轮（说是了离开了考场，其实大概就是在讲话吧。。。）然后安排到了下午考试。
    • 冲上坡道，轮胎大概好像可能要停在坡道实线，（对了好像需要右轮靠边30cm？？？
    • 然后熄火，拉手刹，在松手刹，点火、挂档、慢松离合、拉高转速、最后松开刹车，走！

    考试结束 #

    回考场打印什么什么什么。上午考试下午拿证，下午考试，明天拿证。现在人人都能拿证了，但是路上就还是很少看见电动车变道前会打灯的。

    下周我也要去考试了，先回忆下，补考一次好像70？

    补考确实是70，车管所交钱蛮难的，一个窗口，然后盐都缴费都在那里早上八点四十五到，九点二十才交上钱进去补考。

    拿证了！
