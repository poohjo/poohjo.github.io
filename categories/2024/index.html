    本文总计 0.05k 字, 阅读约需要 0.47 分钟

    hugo, daily

    为什么我要写博客呢？ # 2023年在家躺了一年，感觉时间过得太快且没有任何事情被记录下来，从去年开始自己记忆力下降很快，常常下午就忘记了上午做了什么。脑袋经常性发木，持续时间还久。 24年就开始想着写日记，无奈写在纸上本就难检索，外加长时间没有写过字，写出的字太过鬼画符了，遂想着上网找地方写日记。试过在Word上，如果不新建一个doc，单文件过大且自己需要个类似于Tyora类似的目录结构，遂放弃。 在网上找了许久，找到个这位 「椒盐豆豉」的博客，尤其是这样的book结构太方便我来查看,但是对于前端代码真的一点不懂不知道怎么改，好在自己高中时有捣鼓过点Chrome浏览器,了解点网页是从html渲染来的，但是也就知道这个。后面遂通过F12键显示源文件，加ChatGPT帮助下，扒下来大概结构。 Hugo 安装 # 大概结构是有了，无奈只能本地访问html文件且格式还无法显示，乱七八糟的，可能是扒站方式不对。倒是这个时候，利用ChatGPT得知了Hugo的大概建站过程，同时需要感谢 「椒盐豆豉又双叒叕一篇小白个人博客选什么工具或平台」中有 Hugo建站过程,很重要的是留下了为什么选择Hugo的原因，直接避免了我选择Wordpress。 @ 椒盐豆豉 ：建站不可能三角.jpg 安装基本参考了 「小绵尾巴」中的建站过程，当时遇见蛮多问题，后续通过面向ChatGPT，总算是能有html大概结构出来，但是一直出现一个问题本地打开 html，出现乱码，打开console中一堆爆红，本来都打算放弃了真的因为不知道去哪搞，仔细看看了也很绝望大概就是file:///xxxx/xxxxx.svfg类似这样的爆红，这个时候就要吐槽ChatGPT 3.5 了，就只能提示什么file:///这里应该为file://这样的错误，当时就怀疑是不是哪里扒站的时候多了个斜杠，用Vscod全局搜索了下，没找到三个斜杠地方。。。 都开始试着用Notion来写日记了，无奈那个响应速度确实慢（可能是我主机不行，或者网络不太好。。 后面试着先搞GitHub Pages,文件上传后居然正常显示了，查看了下本来错误的地方现在变成https://就这样变成两个杠，解决。 工具有了，开始记录 # 这个时候看见了 「潮流周刊」这种样式，遂开始装修Hugo。诶，这个时候其实还没开始用Hugo记录几篇，markdown文本格式才刚上手写。但是无奈我引用的图片格式实在太丑，开始了解shortcode 参考的教程 # Hugo | 在 Stack 主题上可行的短代码们 Hugo Shortcodes 示例 博客重启 2.0 尚缺功能 # 能督促自己有记录习惯，暂时还没这个功能，也不知道怎么弄。 图片显示，写了三篇，图片显示全靠现查找 shortcode，大概样式需要重建一下，这个其实我只需要pc端打开不要丑到我真的受不了就行。 评论，这个功能后续再看吧。

    清明放假，第二天了才想起来上次写周刊还是上月初，实在是不忍心在拖下去了来补充一下。 三月还是一样待在家里，没怎么出门，周刊也没有写= - =！实际上也不知道干嘛了。 吃 # 前几天必胜客有自助餐，等我打开app看，盐城地区已经没有票了，后来打电话要到一个到店支付名额遂去。真的吃爽了（可惜那天路考科目挂了。。。。第一次死在了起步熄火了扣10分坡道起步扣10分，第二次走到绕桩被人工判定负，说是起步没按喇叭导致流程还没开始又就这样两条命都没了，十天后才能继续考，今天看了一下应该清明结束就可以去重考了。。。。 清明祭祖 # 今年祭祖任务比较重，因为需要首先要去把祖先牌接到家里，明年祭祖就轮到我家了。另外因为队里要拆迁了，家族里人今年都赶回来烧纸，到公墓里人比往年要多。 往年都是爸爸和我两个人，最多再加上妈妈，今年又去接上了大爷爷、大奶奶，二老下面没有宝宝了，本来有个坤子大爷，年前的时候杀了人被抓起来，枪毙了。之后每年回去要先去接上他们二老一块回去了。 银元被偷 # 老房子梁上一直有用红绿布包着的两块银元，今年回去发现被偷了 奶奶走后，老房子就一直没有住人，村里xun大爷爷（一直也不知道是哪个xun）说家里想借客厅放东西索性就把家门钥匙放在大爷爷那里。清明回去发现，之前一直挂在梁上用红绿布包着的银元没了。诶，唯一的钥匙在大爷爷那里，妈妈那里甚至都没有钥匙，再说平常溜门撬锁的人怎么注意到哪个梁上呢？ 大爷爷也没说什么，就说了我总不能拿吧。。。诶，不到要拆房子时候，平常没人注意到放在梁上用来镇宅的银元。 家里还陆续丢了铜炉，小时候冬天奶奶都用这个铜炉来装炉灰取暖，爸爸说可能还有什么也没注意到。 电影 # 平常无事可看。蛮牛逼，最后教堂那段屠杀戏看得真爽，阮经天的演技撑起了这部戏。 这部还是公映的第一天去看，看了豆瓣上评价却是拉了，人物没立住。倒是看电影的时候隔壁座看到皇帝跪倒在保罗前面时候嘀咕了句：“升官发财死老婆”真的逗乐了。 其实还看了、一年又一年、黑冰、羊城暗哨、沙家浜都蛮一般。我可能是先看了《过把瘾》里的王志文，再看黑冰就真的看不下去了，一年又一年就完全是看作女和直男斗法。 另外可以说的是杨德昌导演的麻将，可惜看到的是巨糊的片源，最后感觉，骗人的人骗来骗去最后还是要骗到自己头上。
